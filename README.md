# MinhaMente

## Inicialização do projeto
```
npm install
```

### Compila e faz hot-reloads para desenvolvimento
```
npm run serve
```

### Compila e minifica para produção
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
